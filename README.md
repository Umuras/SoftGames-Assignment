Game Project (PixiJS)

Description:
This project contains a set of mini-games built with HTML5 & PixiJS. Players can select games from the main menu, and each game includes a pause menu overlay.

Games Included:

Ace of Shadows – Card stack movement game.

Magic Words – Dialogue and avatar-based mini-game.

Phoenix Flame – Simple particle/fire effect game.

⚡ Features

Main Menu: Game selection buttons, animated background effects.

Pause Menu: Overlay for each game; does not stop game layers.

Layer Management: clearGame() function clears previous game objects before starting a new game.

Audio: Background music in the main menu with toggle on/off button.

🛠 Installation & Running

Make sure Node.js and npm are installed.

Open terminal in project folder:

npm install

Start the development server:

npm run dev

Open index.html to run the project. Games can be selected from the main menu.

🎮 How to Play

Click a game button in the main menu to start a game.

Click the pause button in the top-right corner to open the pause menu.

From the pause menu, you can resume the game or switch to another game.

In Magic Words, press SPACE to advance the dialogue.
